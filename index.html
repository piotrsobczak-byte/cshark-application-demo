TeleRecepcja

Production SaaS

Voice-first CRM for auto workshops. A mechanic with dirty hands doesn't need to type anything. The full pipeline: phone call → transcription → structured JSON extraction → data lands in CRM. Client name, car details, job description. All extracted automatically from a single conversation. SMS with magic link lets the mechanic review and confirm on a React dashboard. Full-stack: Python FastAPI + React 19 SPA. Multi-tenant, RODO-compliant, with a prioritized backlog managed in iterative sprints.

Voice-first CRM dla warsztatow samochodowych. Mechanik z brudnymi rekami nie musi niczego pisac. Pelny pipeline: rozmowa telefoniczna → transkrypcja → strukturalny JSON (extraction) → dane trafiaja do CRM-a. Imie klienta, dane auta, opis zlecenia. Wszystko wyciagane automatycznie z jednej rozmowy. SMS z magic linkiem pozwala mechanikowi przejrzec i potwierdzic na dashboardzie React. Full-stack: Python FastAPI + React 19 SPA. Multi-tenant, zgodny z RODO, z priorytetyzowanym backlogiem zarzadzanym w iteracyjnych sprintach.

Car Flip Hunter

Autonomous Bot

4-stage pipeline: scrape → filter/margin → deep analysis → Telegram alert. Scrapes OTOMOTO every 30 min, scores via custom pricing matrix (416 entries: engine, km, year, body type). Price tracking: detects price drops, flags "motivated sellers" (2+ drops/week). AI extracts 35 fields from listings (Gemini 3 Flash), with regex fallback. Scam detection, dealer marking, freshness scoring. 13+ car models, 7 brands tracked.

Sledzenie cen: wykrywa obnizki, flaguje "zmotywowanych sprzedawcow" (2+ spadki/tydzien). AI wyciaga 35 pol z ogloszen (Gemini 3 Flash), z regex fallbackiem. Wykrywanie scamow, oznaczanie dealerow, scoring swiezosci. 13+ modeli, 7 marek sledzonych.
